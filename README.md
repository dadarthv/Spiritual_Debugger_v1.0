# 🛡️ Spiritual Debugger v1.0

> **"For God is not a God of disorder but of peace."** — 1 Corinthians 14:33

這是一個基於 Python 的輕量級心靈辨識工具。旨在幫助使用者（特別是追求真理與秩序的「天國公主」們）過濾紛雜的靈性資訊，確保每一份啟發都建立在聖經的根基與內心的平安之上。

## 🌟 核心理念 (Vision)

在資訊爆炸與神祕主義盛行的時代，我們需要一套「屬靈的防火牆」。本專案將複雜的靈性辨識轉化為結構化的代碼邏輯，實踐「技術 × 人性 × 意義」的交會。

- **身分錨點 (Anchor)**：確認身分，杜絕以知識換取寵愛的焦慮。
- **秩序優先 (Order First)**：透過單元測試 (Unit Test)，確保領悟不偏離現實生活與工作。
- **平安指標 (Peace Metric)**：核心 API 驗證，僅保留指向愛與平安的訊息。

## 🚀 如何運行 (Getting Started)

本工具特別為 Windows 使用者優化，提供「一鍵啟動」功能：

1. **下載專案**: 
   點擊頁面右上方綠色的 `Code` 按鈕，選擇 `Download ZIP` 並解壓縮到您的電腦。

2. **啟動工具**:
   進入資料夾，找到 `啟動我的靈性過濾器.bat` 檔案，雙擊滑鼠左鍵即可啟動。
   *(註：請確保您的電腦已安裝 Python 3.x)*

3. **開發者模式 (指令啟動)**:
   如果您偏好使用終端機，請在專案目錄下執行：
   ```bash
   python Spiritual_Debugger.py

### 💡 為什麼要這樣改？

1. **符合現實情境**：原本的 `git clone` 指令是給已經安裝了 Git 工具的開發者用的。對於追求「秩序與簡便」的妳來說，`Download ZIP` 加上雙擊 `.bat` 檔才是最真實的操作流程。
2. **層次感**：我們保留了「開發者模式」的指令，這展現了妳專案的專業性，同時照顧到了不同層次的使用者。
3. **消除門檻**：這樣別人一看到妳的專案，就知道這是一個「下載就能用」的實用工具，而不是冷冰冰的代碼庫。
